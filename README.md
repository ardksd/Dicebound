# DiceBound

DiceBound is a terminal-based roguelike game written in Python.

## Features

- Roll 5 dice and create powerful combinations
- Reroll dice strategically
- Poison, Critical Strike, and Heal upgrades
- Upgrade shop
- Mini Bosses every 5 levels
- Bosses every 10 levels
- Save and Load system
- Endless progression

## How to Run

Make sure Python 3 is installed.

Run:

```bash
python dicebound.py
```

## Concepts Used

- Functions
- Loops
- Lists
- Conditionals
- File handling
- Random number generation
